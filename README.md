# 🧪 Chemy – AI Powered VR Chemistry Lab

Chemy is a Final Year Engineering Project that provides an AI-powered Virtual Reality Chemistry Laboratory.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- React Three Fiber
- Three.js
- WebXR
- Node.js
- Express.js
- Supabase
- OpenAI API
